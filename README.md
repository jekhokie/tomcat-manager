# Tomcat::Manager

Provide an API to manage an Apache Tomcat Web Server via the Tomcat Manager.

## Installation

Add this line to your application's Gemfile:

    gem 'tomcat-manager'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tomcat-manager
